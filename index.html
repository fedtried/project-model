About the Project

The aim of this report is to conclude whether the introduction of automated robotics would be beneficial to subsea cable maintenance for wind farms in the UK, specifically to the London Array. The report will use cost modelling developed for current operations to perform “what-if” costing analysis to consider different outcomes of the Levelized Cost of Electricity (LCOE) depending on varying levels of automation of subsea vehicles for O&M of subsea cables.

Three scenarios have been explored: Divers, ROV and AUV. The difference between ROV and AUV has been explored throughout the literature review in the final report. AUV technology is still progressing, and as a result has high initial costs. However, it provides increased flexibility compared to the ROV. For more information, see the literature review.

The assumptions used for this model:

1. Each year the capital repayment is kept the same plus interest accrued

2. Windfarm remediation costs and residual value is 0

3. Costs are fixed throughout the lifetime of the offshore wind farm.

Instructions for the model

This model allows the user to change various inputs in order to explore the impact of various variables on the Levelised Cost of Electricity for an offshore windfarm. This model should be used in junction with the final report written where each scenario is fully explored with varying sensitivity analysis cases.

Below are various input boxes alongside the values that are true for the chosen case study of the London Array. The values in the input boxes can be changed to explore it's impact on the LCOE. Additionally, by allowing advanced settings the characteristics of the Offshore Windfarm Array can be edited. Where advanced settings is not selected, the model will use the default values.

The ouputs from the model show the intermediate calculations performed behind the scenes followed by 2 pie charts. The results have been rounded to 2 significant figures as the input values cannot provide improved accuracy. The first pie chart explores the breakdown of OPEX costs. As this model explores using robotics for subsea maintenance, BOP is the variable figure. The second pie chart shows the breakdown of the lifetime costs. In line with literature, the O&M costs account for 25-30% of the total lifetime costs.
